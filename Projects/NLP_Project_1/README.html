# Natural Language Processing

Create a Python project to use various Python libraries to process unstructured text from APIs. The goal is to create senitimental analys of the text information and determined if the text information is negative or positive. Also, create word clouds.

Programming Lanuage

  • Python
  • R interface

Data Cleaning

Processing of text data

  1. remove stop word
  2. lowercase all text
  3. remove stop words
  4. remove special characters
  5. remove punctuation
  6. remove common words
  7. remove rare words

NLP Analysis

  1. Word Frequency: number of words
  2. Word Length: average length of individual words in a text
  3. Lexical Diversity: number of unique words used
  4. Lexical Density: number of unique words divided by the total number of words
