Weather data is essential information that helps in safeguarding life and property. Agriculture depends on knowledge about temperature and precipitation in determining when is the best time to harvest and plant crops. Commodity and stock markets are affected by changes in the adverse weather forecast. There are few organizations or government agencies that don't depend on reliable weather forecasts. Therefore, it is vital to collect and visualize weather information to provide competitive insights for these organizations.

Weather Visualization Tool

Instructions

  • Select one or more cities from the tree view widget GUI.
  • List view widget will populate with the selections
  • Click on the "Run Forecast" button to generate a cities weather forecast.
  • Below on the tab page, there is a data grid of the return data.
  • Also, there is a visualization of the return data
